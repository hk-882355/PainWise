よくある質問

  • 天気データが更新されない: 位置情報の許可を確認してください。
  • HealthKitが連携できない: iOSの「設定 → ヘルスケア → アプリ」で許可を確認。
  • 分析が出ない: 記録数が少ない場合は分析が表示されません。

プレミアム / 購入

  • 購入/解約: iOSの「設定 → Apple ID → サブスクリプション」
  • 購入が反映されない: アプリ内の「購入を復元」をお試しください。
  • 決済情報はAppleが管理します。

データとプライバシー

記録データは基本的に端末内に保存されます。

プライバシーポリシー をご確認ください。
